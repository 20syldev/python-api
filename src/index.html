        GET /color
        Génère une couleur aléatoire.

        Clés :
            - "hex" : génère une couleur en hexadécimal.
            - "rgb" : génère une couleur en rouge-vert-bleu.



        GET /domain
        Renvoie un nom de site factice, un nom de domain et un TLD aléatoire.

        Clés :
            - "domain" : génère un exemple d'URL.
            - "random_name" : affiche un nom de site.
            - "random_tld" : affiche une extension de site.



        GET /lorem?len={Nombre}
        Affiche le nombre de phrases en "Lorem" que vous entrez en paramètre.

        Utilisation :
            - {Nombre} peut prendre en compte n'importe quel nombre entre 1 et 61.
            - Une option vide ou invalide renverra la première phrase.

        Clés :
            - "text" : affiche le texte de la longueur donnée.



        GET /personal
        Génère une carte bancaire, un cvc, un email, une date d'expiration,
        un nom, un pays, une profession et un numéro de téléphone factice aléatoirement.

        Clés :
            - "carte" : génère des numéros de carte bancaire.
            - "cvc" : génère un cvc de carte bancaire.
            - "email" : affiche un email.
            - "expiration" : génère une date d'expiration de carte bancaire.
            - "nom" : affiche un nom et prénom.
            - "pays" : affiche les initiales d'un pays connu.
            - "profession" : affiche une profession.
            - "tel" : affiche un numéro de téléphone.



        GET /qrcode?url={URL}
        Génère un QR code, redirigant vers un lien.

        Utilisation :
            - {URL} peut prendre en compte n'importe quel lien sur Internet.
            - Une option vide ou invalide renverra une erreur.



        GET /token?len={Nombre}
        Génère une clé (token) de la longueur que vous souhaitez.

        Utilisation :
            - {Nombre} peut prendre en compte n'importe quel nombre entre 12 et 4096.
            - Une option vide ou invalide renverra un token de 24 caractères.

        Clé :
            - "key" : affiche la clé générée.



        GET /username
        Renvoie une liste d'adjectifs, d'animaux, de professions, un nombre aléatoire et un pseudo aléatoire.

        Clés :
            - "adjectif" : affiche la liste d'adjectifs disponibles.
            - "animal" : affiche la liste d'animaux disponibles.
            - "nombre" : affiche un nombre aléatoire situé entre 0 et 99.
            - "profession" : affiche la liste des professions disponibles.
            - "username" : affiche un nom d'utilisateur généré à l'aide des autres clés.
