        GET /captcha?text={Texte}
        Génère une image de captcha avec le texte que vous entrez en paramètre.

        Utilisation :
            - {Texte} peut prendre en compte n'importe quel caractère.
            - Une option vide ou invalide renverra une erreur.



        GET /color
        Génère une couleur aléatoire.

        Clés :
            - "hex" : génère une couleur en hexadécimal.
            - "rgb" : génère une couleur en rouge-vert-bleu.



        GET /domain
        Renvoie un nom de site factice, un nom de domain et un TLD aléatoire.

        Clés :
            - "domain" : génère un exemple d'URL.
            - "random_name" : affiche un nom de site.
            - "random_tld" : affiche une extension de site.



        GET /infos
        Affiche quelques informations sur l'API

        Clé :
            - "endpoints": affiche le nombre d'endpoints disponibles.



        GET /lorem?len={Nombre}
        Affiche le nombre de phrases en "Lorem" que vous entrez en paramètre.

        Utilisation :
            - {Nombre} peut prendre en compte n'importe quel nombre entre 1 et 61.
            - Une option vide ou invalide renverra la première phrase.

        Clés :
            - "text" : affiche le texte de la longueur donnée.



        GET /personal
        Génère une carte bancaire, un cvc, un email, une date d'expiration,
        un nom, un pays, une profession et un numéro de téléphone factice aléatoirement.

        Clés :
            - "card" : génère des numéros de carte bancaire.
            - "cvc" : génère un cvc de carte bancaire.
            - "email" : affiche un email.
            - "expiration" : génère une date d'expiration de carte bancaire.
            - "name" : affiche un nom et prénom.
            - "localisation" : affiche les initiales d'un pays connu.
            - "job" : affiche une profession.
            - "tel" : affiche un numéro de téléphone.



        GET /qrcode?url={URL}
        Génère un QR code, redirigant vers un lien.

        Utilisation :
            - {URL} peut prendre en compte n'importe quel lien sur Internet.
            - Une option vide ou invalide renverra une erreur.



        GET /token?len={Nombre}&type={Type}
        Génère une clé (token) de la longueur que vous souhaitez.

        Utilisation :
            - {Nombre} peut prendre en compte n'importe quel nombre entre 12 et 4096. (Sécurité minimale)
            - {Type} peut contenir les options suivantes :
                - "alpha" : Génère un token uniquement composé de lettres (majuscules et minuscules).
                - "num" : Génère un token composé de lettres (majuscules et minuscules) et de chiffres.
                - "base64" : Génère un token encodé en base64.
                - "hex" : Génère un token hexadécimal, composé de chiffres (0-9) et de lettres (a-f).
                - "num" : Génère un token uniquement composé de chiffres.
                - "punct" : Génère un token uniquement composé de caractères de ponctuation.
                - "urlsafe" : Génère un token qui utilise des caractères sûrs pour les URLs (lettres, chiffres, -, et _).
                - "uuid" : Génère un token sous forme d'identifiant universellement unique.
            - Une option vide ou invalide renverra un token de 24 caractères, avec un type alphanumérique.

        Clé :
            - "key" : affiche la clé générée.

        GET /username
        Renvoie une liste d'adjectifs, d'animaux, de professions, un nombre aléatoire et un pseudo aléatoire.

        Clés :
            - "adjective" : affiche la liste d'adjectifs disponibles.
            - "animal" : affiche la liste d'animaux disponibles.
            - "number" : affiche un nombre aléatoire situé entre 0 et 99.
            - "job" : affiche la liste des professions disponibles.
            - "username" : affiche un nom d'utilisateur généré à l'aide des autres clés.
