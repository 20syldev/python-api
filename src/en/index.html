        GET /algorithms?tool={Algorithm}&value={Value}
        Returns an answer based on the algorithm used and the parameter values.

        Usage:
            - {Algorithm} can contain the following options:
                - "anagram": Returns a boolean with the condition that "value" and "value2" are anagrams.
                - "factorial": Returns the factorial of the number entered as parameter.
                - "fibonacci": Returns a list containing the first x terms of the Fibonacci sequence.
                - "palindrome": Returns a boolean with the condition that "value" is a palindrome.
                - "reverse": Returns the value entered as a parameter, in reverse.
            - {Value} can be words or numbers, depending on the algorithm used.
            - {Value2} can be any words or numbers only in the "anagram" algorithm.
            - An empty or invalid option will return an error.

        Key:
            - "answer": displays the answer.



        GET /captcha?text={Text}
        Generates a captcha image with the text you enter as a parameter.

        Usage:
            - {Text} can be any character.
            - An empty or invalid option will return an error.



        GET /color
        Generate a random color.

        Key:
            - "hex": generates a color in hexadecimal.
            - "rgb": generates a red-green-blue color.



        GET /domain
        Returns a fake site name, a domain name and a random TLD.

        Keys:
            - "domain": generates a sample URL.
            - "random_name": displays a site name.
            - "random_tld": displays a site extension.



        GET /infos
        Displays some information about the API

        Key:
            - "endpoints": displays the number of available endpoints.



        GET /lorem?len={Number}
        Displays the number of "Lorem" sentences you enter as parameters.

        Usage:
            - {Number} can be any number between 1 and 61.
            - An empty or invalid option will return the first phrase.

        Keys:
            - "text": displays text of the given length.



        GET /personal
        Generates a credit card, cvc, email, expiration date,
          a name, a country, a job and a random fake phone number.

        Keys:
            - "card": generates credit card numbers.
            - "cvc": generates a credit card cvc.
            - "email": displays an email.
            - "expiration": generates a credit card expiration date.
            - "name": displays a first and last name.
            - "localisation": displays the initials of a known country.
            - "job": displays a job.
            - "tel": displays a phone number.



        GET /qrcode?url={URL}
        Generates a QR code, redirecting to a link.

        Usage:
            - {URL} can be any link on the Internet.
            - An empty or invalid option will return an error.



        GET /token?len={Number}&type={Type}
        Generates a token of any length.

        Usage:
            - {Number} can be any number between 12 and 4096. (Minimum security)
            - {Type} can contain the following options:
                - "alpha": Generates a token containing only letters (upper and lower case).
                - "alphanum": Generates a token containing letters (upper and lower case) and digits (0-9).
                - "base64": Generates a base64-encoded token.
                - "hex": Generates a hexadecimal token, composed of digits (0-9) and letters (a-f).
                - "num": Generates a token containing digits only.
                - "punct": Generates a token containing punctuation characters only.
                - "urlsafe": Generates a token that uses safe characters for URLs (letters, digits, -, and _).
                - "uuid": Generates a token as a unique universal id.
            - An empty or invalid option will return a 24-character token, with an alphanumeric type.

        Key:
            - "key": displays the generated key.

        GET /username
        Returns a list of adjectives, animals, jobs, a random number and a random username.

        Keys:
            - "adjective": displays the list of the available adjectives.
            - "animal": displays a list of the available animals.
            - "number": displays a random number between 0 and 99.
            - "job": displays a list of the available professions.
            - "username": displays a username generated using the other keys.
